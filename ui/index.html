Ownership boundaries

What happens in Ledger

Ledger owns semantic generation

Authority meaning, preview text, publication implications, registry display posture, and bundle composition are generated by Ledger code.

Ledger does not execute authority

This workspace does not call Guard, simulate admissibility, mutate evidence, publish to Cloud, or change runtime outcomes.
